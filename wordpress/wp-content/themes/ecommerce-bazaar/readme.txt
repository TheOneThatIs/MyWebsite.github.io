== Description ==

Ecommerce Bazaar is a modern, fully responsive WordPress theme designed for online stores of any kind, from fashion and electronics to home goods and digital products. With seamless WooCommerce integration, it offers a sleek, professional layout for showcasing products, supporting features like customizable product grids, filters, and secure checkout. Perfect for creating a user-friendly shopping experience, this theme ensures your store looks great across all devices.

= Installation =
	
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in ecommerce-bazaar in the search form and press the 'Enter' key on your keyboard.
3. Click on 'Install' button to install and 'Activate' button to use.
